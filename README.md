Ước mơ trở thành Lập trình viên thông thạo 7=), tôi đã và đang bồi dưỡng những kiến thức về lập trình(Pascal không quan tâm). Chiều cao m66, cân nặng 52kg, không đẹp trai đâu đừng nói nhiều.
